# VLM Dev Environment Context

## Build Requirements
- JDK 17
- Android SDK 35
- Gradle 8.9
- AGP 8.7.0
- Kotlin 2.0.0
- Min SDK 24 / Target SDK 35

## Line Endings
All files use LF (Unix). Windows git is configured with core.autocrlf=false.
Do NOT convert to CRLF — safe for both Termux and Windows 11.

## Key Paths (Termux)
- Repo: ~/projects/VLM
- SDK: ~/android-sdk
- Gradle cache: ~/.gradle

## Key Paths (Windows 11)
- Repo: C:\Users\thoma\projects\VLM (git clone same repo)
- SDK: C:\Users\thoma\AppData\Local\Android\Sdk
- JAVA_HOME: C:\Program Files\Eclipse Adoptium\jdk-17
